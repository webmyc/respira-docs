# Oxygen Builder Reference

Oxygen stores layouts as structured builder data (component trees). Respira can extract and update targeted components while preserving the rest of the page.

- "Update the Oxygen component labeled 'Footer Email' to our new email"
- "Extract Oxygen structure for the homepage"
