# Gutenberg (Block Editor) Reference

Gutenberg stores content as HTML with block comments that represent block structure. Respira can read, update, and analyze Gutenberg pages effectively.

## Best Use Cases

- Editing headings and body content
- Adding/removing sections (blocks) via content updates
- SEO and readability workflows

## Example Prompts

- "Update the first H1 on the About page"
- "Add a new FAQ section at the bottom of page 42"
- "Analyze SEO on my homepage"
