## Welcome to Vibe Coding

Vibe Coding is Respira's innovative WordPress plugin that empowers content creators and developers to safely edit WordPress sites using advanced AI coding assistants like Cursor and Claude Code. Unlike traditional methods that require SSH access and pose security risks, Vibe Coding provides a secure, user-friendly interface for making code changes directly through your WordPress dashboard.
