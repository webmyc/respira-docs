# Full Site SEO Audit Workflow

Run a comprehensive SEO audit using AI and Respira.

## Step 1: List All Pages

Use [wordpress_list_pages](/tools/pages/list-pages) to get all pages:

```text
List all published pages on my WordPress site
```

Note the page IDs for important pages (homepage, services, about, contact).

## Step 2: Analyze Key Pages

```text
Analyze SEO on the homepage
Check SEO issues on the services page
Run SEO analysis on page 42
```

## Step 3: Review Results

Each analysis returns:

- Score (0-100) and Grade (A-F)
- Issues with severity and fix suggestions
- Recommendations prioritized by impact
- Detailed metrics

## Step 4: Prioritize Fixes

**Fix order:**

1. Critical: Missing meta titles, no H1
2. High: Missing meta descriptions, multiple H1s
3. Medium: Images without alt text, thin content
4. Low: Schema markup, internal linking

## Step 5: Apply Fixes

```text
Update the homepage meta description to: Your compelling description here
Add alt text to all images on the services page
Add an H1 heading to the about page
```

## Step 6: Verify

Re-run analysis to confirm improvements:

```text
Analyze SEO on the homepage again
```

## Batch Audit Prompt

```text
Run SEO analysis on these pages and give me the top 5 issues across all of them:
- Homepage (ID: 2)
- Services (ID: 15)
- About (ID: 20)
- Contact (ID: 25)
```
