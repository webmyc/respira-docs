## Installation Overview

Installing Vibe Coding is a straightforward process that takes just a few minutes. Our plugin is designed for easy integration with existing WordPress sites, whether you're running a single blog or managing multiple wisdom economy teaching platforms.

<Steps>
  <Step title="Download Plugin" icon="download">
    Visit the WordPress plugin repository or download Vibe Coding directly from our website.
    
    <CodeGroup tabs="WordPress Admin,Manual Upload">
      ```bash
# Via WordPress Admin
1. Go to Plugins > Add New
2. Search for "Vibe Coding"
3. Click Install Now
      ```
      ```bash
# Manual Upload
1. Download vibe-coding.zip
2. Go to Plugins > Add New > Upload Plugin
3. Select the ZIP file and click Install Now
      ```
    </CodeGroup>
  </Step>
  <Step title="Activate Plugin" icon="check-circle">
    After installation, activate Vibe Coding from the Plugins page.
    
    <Callout kind="info">Activation adds the Vibe Coding menu to your WordPress admin sidebar.</Callout>
  </Step>
  <Step title="Configure Settings" icon="settings">
    Navigate to Vibe Coding > Settings to configure your AI assistant connections.
    
    <Expandable title="Required Settings" default-open="false">
      - API endpoint URL
      - Authentication credentials
      - File permissions
      - Backup preferences
    </Expandable>
  </Step>
  <Step title="Test Connection" icon="wifi">
    Verify your AI assistant connection and run a test edit.
    
    <CodeGroup tabs="Cursor,Claude Code">
      ```javascript
// Test with Cursor
console.log('Testing Vibe Coding connection...');
// Expected output: Connection successful
      ```
      ```python
# Test with Claude Code
print("Testing Vibe Coding integration...")
# Expected output: Integration verified
      ```
    </CodeGroup>
  </Step>
</Steps>

## Troubleshooting Installation

<ExpandableGroup>
  <Expandable title="Plugin Won't Activate" default-open="false">
    Check PHP version compatibility and ensure all required extensions are installed.
    
    ```bash
# Check PHP version
php --version
    ```
  </Expandable>
  <Expandable title="Connection Failed" default-open="false">
    Verify API credentials and firewall settings. Ensure outbound connections are allowed.
    
    ```bash
# Test connectivity
curl -I https://api.vibe-coding.com
    ```
  </Expandable>
  <Expandable title="Permission Errors" default-open="false">
    Adjust file permissions for the wp-content directory and plugin folders.
    
    ```bash
# Fix permissions
chmod 755 wp-content/
chmod 644 wp-content/plugins/vibe-coding/*
    ```
  </Expandable>
</ExpandableGroup>

<Callout kind="alert">If you encounter persistent issues, contact our support team with your WordPress version and error logs.</Callout>
