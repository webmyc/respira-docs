# Contact

If you need help with Respira for WordPress, start with the docs and troubleshooting, then contact support with the details needed to resolve the issue quickly.

## Start here

- [Troubleshooting](/troubleshooting)
- [Configuration](/configuration)
- [Tools Reference](/tools/overview)

## Email support

Email: [word@respira.press](mailto:word@respira.press)

## What to include

To help us debug quickly, include:

- Your site URL (or staging URL)
- Respira plugin version
- MCP server version
- What you tried (the exact prompt/tool call)
- The error message / error code (see [Error Codes](/reference/error-codes))
- Screenshots (if UI-related)
