# wordpress_create_post_duplicate

Create a duplicate of an existing post for safe editing. After creating a duplicate, edit the duplicate using [wordpress_update_post](/tools/posts/update-post), then approve in WordPress Admin (Respira â†’ Approve Edits).

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| originalId | number | Yes | Original post ID |
| suffix | string | No | Optional suffix |

## Response

```json
{
  "success": true,
  "original_id": 101,
  "duplicate_id": 202,
  "status": "draft",
  "approval_url": "https://yoursite.com/wp-admin/admin.php?page=respira-approvals"
}
```
