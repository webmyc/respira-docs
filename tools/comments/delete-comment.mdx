# wordpress_delete_comment

Delete a comment.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| id | number | Yes | Comment ID |

## Response

```json
{
  "success": true,
  "deleted": true,
  "id": 9001
}
```

## Example Prompts

- "Delete comment 9001"
- "Remove the spam comment with ID 9001"
