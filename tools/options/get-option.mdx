# wordpress_get_option

Get option value by name.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| option | string | Yes | Option name |

## Response

```json
{
  "success": true,
  "option": "blogname",
  "value": "My Site"
}
```

## Example Prompts

- "Get the blogname option"
- "What is the value of option 'permalink_structure'?"
