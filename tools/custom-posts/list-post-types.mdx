# wordpress_list_post_types

List all registered post types.

## Parameters

This tool takes no parameters.

## Response

```json
{
  "success": true,
  "post_types": [
    {
      "type": "page",
      "label": "Pages"
    },
    {
      "type": "post",
      "label": "Posts"
    }
  ]
}
```

## Example Prompts

- "List post types"
- "What custom post types are available?"
