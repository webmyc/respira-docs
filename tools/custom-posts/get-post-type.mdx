# wordpress_get_post_type

Get post type details.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| type | string | Yes | Post type name |

## Response

```json
{
  "success": true,
  "type": "product",
  "label": "Products",
  "public": true
}
```

## Example Prompts

- "Get post type details for 'product'"
- "Tell me about the 'event' post type"
