# wordpress_delete_custom_post

Delete a custom post.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| type | string | Yes | Post type name |
| id | number | Yes | Post ID |

## Response

```json
{
  "success": true,
  "deleted": true,
  "type": "product",
  "id": 3001
}
```

## Example Prompts

- "Delete product 3001"
- "Remove portfolio item 42"
