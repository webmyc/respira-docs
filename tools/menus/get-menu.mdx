# wordpress_get_menu

Get a navigation menu with all its items. Includes WPML translation info if available.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| id | number | Yes | Menu ID |

## Response

```json
{
  "success": true,
  "menu": {
    "id": 10,
    "name": "Primary Menu",
    "items": [
      {
        "id": 1001,
        "title": "Home",
        "url": "/",
        "parent": 0
      }
    ]
  }
}
```

## Example Prompts

- "Get menu 10"
- "Show me the items in the primary menu"
