# wordpress_get_menu_item

Get a single menu item by ID.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| item_id | number | Yes | Menu item ID |

## Response

```json
{
  "success": true,
  "item": {
    "id": 1001,
    "title": "Home",
    "url": "/",
    "parent": 0
  }
}
```

## Example Prompts

- "Get menu item 1001"
- "Show details for the menu item that links to /pricing"
