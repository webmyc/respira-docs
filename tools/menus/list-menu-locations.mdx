# wordpress_list_menu_locations

List all registered menu locations and their assigned menus.

## Parameters

This tool takes no parameters.

## Response

```json
{
  "success": true,
  "locations": [
    {
      "location": "primary",
      "description": "Primary Menu",
      "menu_id": 10
    }
  ]
}
```

## Example Prompts

- "List menu locations"
- "Which menu is assigned to the primary navigation?"
