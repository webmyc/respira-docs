# wordpress_list_menus

List all navigation menus. Includes WPML translation info if available.

## Parameters

This tool takes no parameters.

## Response

```json
{
  "success": true,
  "menus": [
    {
      "id": 10,
      "name": "Primary Menu"
    }
  ]
}
```

## Example Prompts

- "List all menus"
- "What navigation menus exist on my site?"
