# wordpress_get_taxonomy

Get taxonomy details.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| taxonomy | string | Yes | Taxonomy name |

## Response

```json
{
  "success": true,
  "taxonomy": {
    "name": "category",
    "label": "Categories",
    "public": true
  }
}
```

## Example Prompts

- "Get taxonomy details for 'category'"
- "Tell me about the 'product_cat' taxonomy"
