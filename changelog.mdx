## Installation instructions

### WordPress plugin

1. Download the latest plugin ZIP file from the release list above.
2. Go to WordPress Admin → Plugins → Add New → Upload Plugin.
3. Upload the ZIP file and click Install Now.
4. Activate the plugin.
5. Generate your API key from the admin menu under Respira → API Keys.

### MCP server

1. Download the latest MCP server archive.
2. Extract the archive to your desired deployment location.
3. Configure your AI coding assistant to connect to the MCP server endpoint.
4. For detailed setup steps, see the [Installation guide](/installation).
